Demonstration of the Median as the Minimum Sum of Absolute Deviations

Let \( X = \{x_1, x_2, ..., x_n\} \) be an ordered set of values, and we seek a value \( c \) that minimizes the sum of absolute deviations:

\[ S(c) = \sum_{i=1}^{n} |x_i - c| \]

To prove that the median is the value of \( c \) that minimizes \( S(c) \), we can reason as follows:

  • Continuity: The absolute error function \( |x_i - c| \) is continuous and linear between the values \( x_1, x_2, ..., x_n \).
  • Derivative: To minimize \( S(c) \), we can study the derivative of the sum of absolute deviations with respect to \( c \). However, the derivative of the absolute value function does not exist at the exact points \( c = x_i \), but the slope of the function changes sign at the points where \( c \) crosses a value \( x_i \).
  • Median: The value of \( c \) that minimizes the sum of absolute deviations is found when there is a balance in the number of values above and below \( c \), and this is exactly what the median represents.

Intuitively, the sum of absolute deviations is minimized when a value is chosen that "divides" the dataset into two equal parts in terms of the number of data points. This is exactly what the median does.

Other Ways to Define Central Tendency (Location Statistics)

Apart from the median, there are many ways to define a measure of position or central tendency for a distribution. Below are some common definitions, each with its generalization and motivation:

  • Arithmetic Mean: The mean is the sum of all values divided by the total number of observations. It is the most common measure of centrality and has useful mathematical properties, such as minimizing the sum of squared deviations: \[ \text{Mean} = \frac{1}{n} \sum_{i=1}^{n} x_i \]
  • Mode: The mode is the value that appears most frequently in a dataset. It is a useful measure when trying to identify the most "popular" value in a dataset.
  • Geometric Mean: This is used for distributions with growth rates or when the data are multiplicative, such as interest rates or economic growth: \[ \text{Geometric Mean} = \left( \prod_{i=1}^{n} x_i \right)^{1/n} \]
  • Harmonic Mean: The harmonic mean is useful in situations where the data represent ratios, such as speeds or rates: \[ \text{Harmonic Mean} = \frac{n}{\sum_{i=1}^{n} \frac{1}{x_i}} \]
